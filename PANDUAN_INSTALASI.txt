

PANDUAN INSTALASI SOURCE CODE

1. Buka localhost/phpmyadmin pada web browser.
2. Buat database baru dengan nama 'db_alifqy'.
3. Import file 'db_alifqy.sql' ke database 'db_alifqy' yang sudah dibuat.
4. Copy folder 'alifqysport' ke folder htdocs. 
	Letak folder htdocs XAMPP windows ada di C://XAMPP/htdocs
5. Akses toko online dengan alamat 'localhost/alifqysport pada web browser.
6. Selesai.
